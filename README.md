## Good to know (remember)
In order to import library to your python project follow next instuction. [Here is video version of them.](https://www.youtube.com/watch?v=ThU13tikHQw)

Perform all steps in the terminal
1. Type ```python -m venv (path to the project)\venv```
2. Type ```Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process```
3. Execute ```venv\scripts\activate```
4. Type ```pip install (name of your library)```
